Project Overview

What is WeatherNow?

WeatherNow is a lightweight web dashboard that fetches live weather data and 7-day forecasts for any city. Its main purpose is to demonstrate API integration, responsive UI, and CI/CD deployment in a student project.

Target audience: Instructors, classmates, and anyone who needs a clean, quick weather snapshot.

Key Goals

  • Consume external weather API (OpenWeatherMap)
  • Responsive design for mobile & desktop
  • Automated build & deploy with GitHub Actions → Netlify/Vercel

Success Criteria

  • Accurate current weather & 7-day forecast
  • Accessible UI, loading and error states
  • Public demo link + GitHub repo with CI/CD
